Detailed Features of SymiSun* 03

By Stam, October 2nd, 2009 in News / Announcements

Let's take a look at the built-in features of this new template:

  • Super fast loading, overall size of template images being 40kb
  • Cross-browser compatible, the layout remains consistent in every modern browser
  • W3C standards compliant, valid xHTML 1.0 Strict and CSS 2.1
  • Search Engine Friendly, using HTML tags in proper places and order
  • Content rich header, including User Panel, Site Info and Recent Items with appealing tabs
  • Clean tableless interface with large fonts and plenty of white space
  • Custom icon set, specially created for this template
  • Sleek rounded buttons with no loading delay on rollover due to the sprite technique
  • Modern footer with search input, important site shortcuts and link to the top of page
